# RabbitMQ-Workers-js

Uma aplicação simples em Node.js de troca de mensagem usando RabbitMQ.


#### Requerimentos
-----------------
* Versão do node.js:

      NODE.JS >= 12.15.0

* Versão do npm:

      NPM >= 6.13.7

* Versão do Docker:

      DOCKER >= 18.06.3
